# BBC Chatbot Project

## Description
This project is a BBC news chatbot / analysis project built with Python.

## Files
- `code.py`: main source code
- `requirements.txt`: required libraries
- `data/`: dataset
- `bbc_chatbot_dynamic/`: project folder
- `bbc.pdf`: report

## How to run
```bash
pip install -r requirements.txt
python code.py
